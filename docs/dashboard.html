How to prepare a payload click to expand · dismiss with ×
  1. Choose your payload. Click Presets ▾ and load a saved template, or build from scratch with + Add entry / + Add layer / + Add pinned. Presets are portable — any chain can load any preset.
  2. Wire entries to sources. Each entry needs at least one source file or directory. Layers consume entries on rotation; pinned positions consume them once at a fixed position.
  3. Apply to chain. Once the draft validates, click Apply to chain. This writes the template into chain.json and records the preset name. Only allowed when the lock badge says EDITABLE — the chain locks while an Age is in progress.
  4. Build the artifacts. Click Build to materialize Payload/<chain>/… from your sources. Rebuild whenever you change a source file — the chain template stays put.
  5. Seal the Age. Click Seal to begin distributing the new payload. Future mints land chunks into this Age’s positions; the config is frozen into the seal record, so editing the preset later won’t affect this Age. To iterate, wait for the cycle to complete, then apply & seal again.
